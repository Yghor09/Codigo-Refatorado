# Projeto de Exemplo

Este é um pequeno projeto web composto por um arquivo HTML (`index_refatorado.html`) e uma folha de estilo CSS (`style.css`).

O objetivo principal é demonstrar a refatoração de um layout simples e aplicar estilos com CSS.

## Arquivos

- `index_refatorado.html` - página HTML principal refatorada.
- `style.css` - estilos aplicados ao HTML.

## Como usar

1. Abra o arquivo `index_refatorado.html` em um navegador web.
2. Confira o layout e os estilos aplicados.
